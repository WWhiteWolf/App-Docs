# Session start — App-Docs (cross-project home)

**At every session start, read: `CLAUDE.md` (this folder),
this file, and honor Patrick's saved Cowork preferences.**

Then:

1. Read `master-handoff.md` — the one session chain for all projects.
   It says where things stand and names the next session's goal.
2. If the session is publishing/business-side, read
   `Publishing-Strategy.docx` (the north star).
3. Connect the project folder the session's goal lives in and read THAT
   project's `CLAUDE.md`, `session-start.md`, and key docs.
4. Read the goal back to Patrick; he confirms or redirects. Scope it
   together; wait for his "go" before changing anything.

The full standing rules live in
`elderlyassistant/docs/session-start.md`; the short form is in
`master-handoff.md` and each folder's `CLAUDE.md`.

At session end: refresh `master-handoff.md` "Where things stand" and the
active project's own docs. Patrick commits everything himself.
